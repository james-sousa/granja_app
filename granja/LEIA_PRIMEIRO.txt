💡 DICAS IMPORTANTES:

1. Arquivo .env é como uma SENHA
   - NUNCA compartilhe com ninguém
   - NUNCA commit para Git (já está em .gitignore)
   - Guarde em lugar seguro

2. Arquitetura Híbrida
   - Supabase + credenciais preenchidas? → usa remoto
   - Sem .env ou credenciais? → usa SQLite local (dev)
   - Permite transição gradual

3. Múltiplos Usuários
   Todos acessam o MESMO banco remoto:
   
   Máquina A (Desktop)  ┐
   Máquina B (Tablet)   ├─ → Supabase → PostgreSQL Brasil
   Máquina C (Web)      ┘
   
   Dados sincronizados em tempo real! ⚡

4. Quando algo der errado
   - Veja logs: granja_manager/logs/app.log
   - Leia SUPABASE_SETUP.md → seção Troubleshooting
   - Teste no Supabase Dashboard → Table Editor

❓ PERGUNTAS FREQUENTES:

P: Preciso pagar para usar Supabase?
R: Não! Plano gratuito inclui 50GB e 1M requisições/mês. Mais que suficiente.

P: E se minha internet cair?
R: Sem Supabase → fallback para SQLite. Você continua trabalhando offline.
