EXEMPLOS DE USO
================================================================================

EXEMPLO 1: Criar um Pedido (Fluxo Obrigatório)
    
    Cliente novo "João Silva" realiza pedido:
    
    1. Sistema verifica: Existe "João Silva"?
    2. Sistema detecta: NÃO EXISTE
    3. Sistema cria: Cliente "João Silva" automaticamente
    4. Sistema valida: Estoque de "Frango (kg)"?
    5. Sistema cria: Pedido com itens
    6. Sistema atualiza: Estoque reduz de 35 para 34
    7. Dashboard atualiza: Vendas hoje aumentam R$ 22.00
    
    TUDO AUTOMATICAMENTE! 🤖

EXEMPLO 2: Marcar Pedido como Pago
    
    1. Cliente "João Silva" paga
    2. Você clica "Marcar pago" no pedido
    3. Dashboard: Vendas de hoje aumentam
    4. Pendências: Desaparecem de "Clientes com pendência"

EXEMPLO 3: Criar Gasto
    
    1. Você compra ração: R$ 320
    2. Clica "Novo gasto"
    3. Inforna: Descrição "Ração", Categoria "Insumos", Valor 320
    4. Clica "Salvar"
    5. Dashboard: Lucro do mês diminui R$ 320

================================================================================
BANCO DE DADOS
================================================================================

Localização:
    granja_manager/data/granja.db

Criado automaticamente na primeira execução

Tabelas:
    - clientes (id, nome, telefone, criado_em)
    - produtos (id, nome, preco, estoque, ativo, criado_em)
    - pedidos (id, cliente_id, data, total, pago, concluido)
    - itens_pedido (id, pedido_id, produto_id, quantidade, preco_unitario)
    - gastos (id, descricao, categoria, valor, data)

Dados persistem 100% - Nada fica em memória!
