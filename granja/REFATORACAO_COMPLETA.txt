📊 FLUXO AGORA:
═══════════════════════════════════════════════════════════════════════════

CADASTRO:
────────
User → "Cadastre-se" 
  ↓
auth_service.criar_usuario(email, nome, senha)
  ↓
supabase.auth.sign_up({
    email, password,
    data: {nome}
})
  ↓
Supabase cria em auth.users (tabela interna)
  ↓
Email de confirmação enviado automaticamente


LOGIN:
──────
User → "Entrar"
  ↓
auth_service.fazer_login(email, senha)
  ↓
supabase.auth.sign_in_with_password({email, password})
  ↓
Supabase verifica + gera JWT token
  ↓
Retorna: {user, session.access_token}
  ↓
Token enviado em Authorization header automaticamente


REQUISIÇÕES AO BANCO:
────────────────────
repositório.listar()
  ↓
supabase.from_('tabela').select('*').execute()
  ↓
Token enviado automaticamente no header
  ↓
RLS valida: WHERE user_id = auth.uid()
  ↓
Apenas dados do usuário retornados


LOGOUT:
───────
User → Ícone vermelho (canto superior direito)
  ↓
on_logout()
  ↓
supabase.auth.sign_out()
  ↓
Sessão invalidada no servidor
  ↓
Redireciona para /login

🚀 COMO USAR:
═══════════════════════════════════════════════════════════════════════════

1. CRIAR .env na raiz do projeto:
   ──────────────────────────────
   
   SUPABASE_URL=https://seu-projeto.supabase.co
   SUPABASE_ANON_KEY=seu_anon_key_aqui
   
   Obter em: https://app.supabase.com → Project → Settings → API


2. EXECUTAR APLICAÇÃO:
   ───────────────────
   
   python run.py


3. TESTAR:
   ───────
   
   1. Clique "Cadastre-se aqui"
   2. Preencha: nome, email, senha (min 6 chars + 1 número)
   3. Clique "Cadastrar"
   4. Volta ao login automaticamente
   5. Digite email e senha
   6. Clique "Entrar"
   7. Acesso ao dashboard!
   8. Nome do usuário aparece no topo
   9. Clique no ícone vermelho para logout
