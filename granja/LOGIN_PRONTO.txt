🚀 COMO TESTAR:
═══════════════════════════════════════════════════════════════════════════

Opção 1: COM SUPABASE (RECOMENDADO)
─────────────────────────────────
1. Configure seu .env com as credenciais do Supabase:
   SUPABASE_URL=sua_url
   SUPABASE_KEY=sua_chave

2. Execute setup_supabase.py:
   python setup_supabase.py

3. Rode a aplicação:
   python run.py

4. Na tela de login:
   - Clique em "Cadastre-se aqui"
   - Crie um novo usuário
   - Volte e faça login com as credenciais


Opção 2: MODO OFFLINE (SQLite local)
─────────────────────────────────────
1. A aplicação usará SQLite automaticamente
   (se .env não estiver configurado)

2. Execute:
   python run.py

3. Cadastre um usuário novo
   Faça login com as credenciais


🔍 TESTES RÁPIDOS:
═══════════════════════════════════════════════════════════════════════════

□ Teste 1: Criar usuário
  - Clique em "Cadastre-se aqui"
  - Preencha todos os campos
  - Clique em "Cadastrar"
  - Esperado: Volta ao login com mensagem de sucesso

□ Teste 2: Login correto
  - Insira email e senha do usuário criado
  - Clique em "Fazer Login"
  - Esperado: Mostra o dashboard com nome do usuário no topo

□ Teste 3: Login incorreto
  - Insira email/senha errados
  - Clique em "Fazer Login"
  - Esperado: Mostra mensagem de erro

□ Teste 4: Logout
  - Logado no dashboard
  - Clique no ícone de logout (canto superior direito)
  - Esperado: Volta à tela de login

□ Teste 5: Protecção de rota
  - Abra URL: http://localhost:8000/dashboard
  - Esperado: Redireciona para /login se não logado
